# Projeto Prático 03 — SPA (Entrega)

Conteúdo:
- SPA simples com sistema de templates (templates `<template>`).
- Formulário com validação de campos e armazenamento em `localStorage`.
- Estilos usando paleta azul / rosa / roxo.
- Arquivos prontos para abrir localmente.

Como usar:
1. Extraia o zip.
2. Abra `index.html` em um navegador moderno.
3. Navegue pelas seções (Início, Projetos, Contato, Sobre).
4. No formulário, preencha e envie — as submissões ficam salvas no localStorage.

Observações:
- Projeto pensado para cumprir as especificações obrigatórias: manipulação do DOM, SPA básico, templates JS e verificação de consistência de dados em formulários.
